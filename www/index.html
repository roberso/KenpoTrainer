Kenpo Trainer 1.0

This is a tool based on Ed Parker's American Kenpo Karate developed by Kenn Roberson, a Kenpo black belt. Material contained in this tool is compiled from the internet, and personal knowledge from years of study in Kenpo.

The programming of this app was entirely done by the author using originally IBM's Mobile First tool and JQuery Mobile 1.4.4 back in 2015. This version version was ported over to Apache Cordova and using JQuery Mobile 1.4.5 to stick to more open source based development environent.

The following are good sites for more information on Kenpo

  • Texas Storm Kenpo
  • The KenpoNet
  • PCWood's Kenpo Site
  • Brian's Parker Kenpo Page
  • Rick Fowler's Kenpo

I hope you enjoy this app.
